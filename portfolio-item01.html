Página genérica de login login-page

Com página para criação de nova conta de acesso

Essa página foi hospedada na plataforma Netfily.app. Seu acesso pode ser realizado através do link a seguir: generic-login-page.netlify.app.

O repositório para este projeto encontra-se hospedado no GitHub. Clique aqui para acessá-lo.


Esse projeto teve como objetivo criar uma página inicial genérica de login.
A imagem de fundo pode ser alterada de acordo com a finalidade de uso.
A imagem da página principal do projeto é mostrada abaixo.

Ao clicar em Registre-se, a página é redirecionada para o formulário de criação de nova conta, conforme demonstrado na imagem a seguir.

Caso o usuário já possua uma conta, ao clicar em Entrar, ele é redirecionado novamente para a página principal.
